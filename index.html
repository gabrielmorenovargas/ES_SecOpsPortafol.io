Sobre mí:

¡Hola y bienvenido a mi Portafolio Profesional en Seguridad de la Información!

Mi nombre es Gabriel Moreno Vargas, soy un profesional apasionado y dedicado en Seguridad de la Información con experiencia comprobada en el área.

Con una sólida formación en TI, Redes, Seguridad de la Información, Seguridad de Redes y Análisis de Amenazas, sigo desarrollando las habilidades y conocimientos para ayudar a las organizaciones a proteger sus activos frente a ciberamenazas.

Gracias por visitar mi portafolio, espero poder conectar contigo.

Mis Proyectos

Proyecto 1: Identificación de Amenazas y Análisis de Efectividad de Controles

Este proyecto consistió en evaluar una aplicación de correo electrónico ampliamente utilizada para determinar si los controles de seguridad existentes mitigaban efectivamente vulnerabilidades conocidas. El análisis se enfocó en identificar amenazas potenciales, mapearlas a medidas de control y evaluar su desempeño. El resultado evidenció brechas de protección y brindó recomendaciones para fortalecer la postura de seguridad de la aplicación.

Proyecto 2: Programa Mensual de Auditoría de TI para una Cooperativa Financiera en Ecuador

Este proyecto consistió en diseñar y ejecutar un programa de auditorías recurrentes para fortalecer la gobernanza y el cumplimiento en una cooperativa financiera. Incluyó la creación de planes de auditoría detallados a nivel de proceso con actividades, acciones y verificaciones. Las auditorías mejoraron la visibilidad de las prácticas operativas y la adherencia a estándares de seguridad y gestión de TI.

Ver Proyecto

Proyecto 3: Informes Integrales de Auditoría sobre Procedimientos y Controles de TI

Este proyecto se enfocó en elaborar informes integrales de auditoría para programas anuales de TI. Los informes evaluaron procedimientos y controles conforme a estándares internacionales como ISO 27001 e ISO 22301, además de normativas nacionales. Se prestó especial atención a la continuidad del negocio y la gestión de respaldos, entregando recomendaciones prácticas para mejorar la resiliencia y el cumplimiento.

Proyecto 4: Laboratorio SIEM (Security Information and Event Management)

Este proyecto implicó la implementación de un laboratorio SIEM utilizando Wazuh para gestionar agentes de seguridad en sistemas Windows 10 y Ubuntu Desktop dentro de entornos controlados con VirtualBox. El laboratorio brindó experiencia práctica en monitoreo centralizado de eventos, análisis de logs y detección de incidentes, simulando entornos empresariales reales.

Proyecto 5: Laboratorio de Emulación de Adversarios con Caldera y Docker

Este proyecto consistió en la configuración de un laboratorio controlado para la emulación de adversarios utilizando Caldera y Docker. Se simularon ataques de reconocimiento en máquinas Windows 10 y Ubuntu Desktop en VirtualBox para evaluar capacidades de detección y respuesta. El laboratorio sirvió como ejercicio práctico en metodologías red-team/blue-team y simulación adversarial para formación en ciberseguridad.
